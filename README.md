# 🔁 Cohort & Retention Analysis

A tool for analysing participant engagement over time using cohort-based retention analysis.

This project is part of the **Impact Analytics Suite**, a portfolio of decision-support tools focused on impact, programme, and data strategy workflows.

---

## 🎯 Overview

Retention analysis helps teams understand how long participants remain engaged and where drop-off occurs. This tool visualises engagement patterns across cohorts.

---

## ✨ Key Features

- Cohort grouping by signup period  
- Retention heatmap  
- Retention curve visualisation  
- Drop-off analysis  
- Cohort comparison  

---

## 🧠 Why I Built It

I built this to demonstrate how time-based analysis can reveal patterns in engagement and help improve programme design.

---

## 👥 Example Use Case

- Tracking engagement across programme cohorts  
- Identifying early drop-off patterns  
- Comparing retention across time periods  

---

## 🗂 Expected Data Structure

- participant_id  
- signup_date  
- activity_date  
- programme  

---

## 🧪 Demo Mode

Includes synthetic cohort data showing realistic retention decay and improved performance in newer cohorts.
